Pages
GET /map
Interactive live map with ADS-B aircraft tracking, LoRaWAN nodes, gateways, and DOA detection visualization.
Node Management
GET /nodes
List all registered LoRaWAN sensor nodes with their positions, ranges, and gateway connections.
POST /nodes
Register a new sensor node.
Body: { dev_eui, name, lat, lon, range, altitude }
PUT /nodes/<dev_eui>
Update an existing node's properties (position, range, name).
DELETE /nodes/<dev_eui>
Remove a node from the system.
Gateway Management
GET /gateways
List all registered LoRaWAN gateways with positions and coverage ranges.
POST /gateways
Register a new gateway.
Body: { gateway_id, name, lat, lon, range, altitude }
PUT /gateways/<gateway_id>
Update a gateway's properties.
DELETE /gateways/<gateway_id>
Remove a gateway from the system.
Data Ingestion
POST /uplink
Chirpstack uplink webhook — receives LoRaWAN sensor data (DOA detections, mic-check heartbeats) and broadcasts via WebSocket.
Body: Chirpstack uplink JSON payload
Detection & Classification
GET /detections/recent
Retrieve the most recent DOA detections from all nodes.
GET /api/sound_classes
List all known sound classification categories used by the detection system.
ADS-B Aircraft Data (Proxied)
GET /api/adsb
Live aircraft positions within range. Proxied from adsb.lol — returns all aircraft near the configured map center.
GET /api/adsb/track/<icao24>
Historical flight trail for a specific aircraft. Uses globe.adsb.lol trace data with OpenSky Network as fallback.
POST /api/adsb/routeset
Aircraft route lookup (origin → destination airports). Proxied from adsb.lol/adsb.im routeset API.
Body: { planes: [{ callsign, lat, lon }] }
